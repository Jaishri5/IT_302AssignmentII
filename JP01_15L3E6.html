Introduction to DCN

The word data refers to information.
Data communication: The exchange of data between two parties via some form of transmission medium.

Network Models

  • OSI Model has 7 layers and given by ISO
  • TCP/IP Model has 5 layers

Physical Layer and DataLink layer

  • Physical layer uses data in form of bits.
  • Data link layer uses data in form of frames.It uses physical address

Internet Layer

  • Internet layer uses data in form of datagram.
  • It uses IP protocol
  • It uses logical or IP address

Transport Layer

  • Internet layer uses data in form of segments/datagram/packets.
  • It uses TCP/UDP/SCTP protocol
  • It uses port address

Application Layer

  • Application layer uses data in form of messages.
  • It uses protocol depending on the service used
  • It has services like Email, File Transfer, DNS etc
